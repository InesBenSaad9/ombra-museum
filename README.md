# Ombra Museum
Ombra Museum is an interactive digital museum experience built with React and TypeScript.  
It allows users to explore artworks, interact with virtual environments, and engage with an AI-powered oracle for immersive storytelling.
## Features

-  Interactive artwork gallery
-  Virtual museum navigation
-  AI Oracle chat (Gemini API)
-  Quiz and leaderboard system
-  Immersive components (carousel, banners, etc.)

## Tech Stack

- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express (in progress)
- API: Gemini AI
## Installation

1. Clone the repository
git clone https://github.com/InesBenSaad9/ombra-museum.git

2. Navigate into the project
cd ombra-museum

3. Install dependencies
npm install

## Environment Variables

Create a `.env` file in the root and add:

GEMINI_API_KEY=your_api_key_here
APP_URL=your_app_url


## Deployment

The frontend can be deployed using Vercel or Netlify.
The backend (if used) can be deployed on Render or Railway.
